Instructions

Important instructions:

Once you begin,
  • Do NOT refresh (Do not hit F5 or the refresh icon or right click)
  • Do NOT press the back button (Do not press the back button or right click)
  • Do take the study only in Google Chrome.
This means that you are requested to preferably use only the left mouse button and only on the actual web page to navigate.
Should you run into any technical problems during the study, please stand up and ask any of the volunteers for help.

Screen Size

Please follow the following instructions for optimal experience
  1. When you go to the practice pages, resize the web page (using Ctrl plus + or -)
  2. When you don't have to scroll anymore, stop resizing
  3. The current page config can act as the base zoom level
  4. Then the pages can be tweaked to suit preferences using the same controls
